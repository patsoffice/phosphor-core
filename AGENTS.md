# AGENTS.md

See `CLAUDE.md` and per-crate `CLAUDE.md` files for all development guidelines. This project uses hierarchical CLAUDE.md files:

- `/CLAUDE.md` - Workspace-wide build, test, commit conventions
- `/core/CLAUDE.md` - CPU implementation rules, flag conventions, testing patterns
- `/core/src/cpu/CLAUDE.md` - Per-CPU file organization and instruction workflow
- `/machines/CLAUDE.md` - Machine/board implementation patterns
- `/frontend/CLAUDE.md` - SDL2 frontend specifics
